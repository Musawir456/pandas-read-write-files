# Reading and Writing CSV & Excel Files in Pandas

This repository focuses on **reading and writing data files** using Pandas, which is essential for importing datasets, saving results, and working with real-world data.

## Topics Covered
- Reading CSV files with `read_csv()`
- Reading Excel files with `read_excel()`
- Writing DataFrames to CSV with `to_csv()`
- Writing DataFrames to Excel with `to_excel()`
- Handling file paths and sheet names
- Basic file input/output options
- Practical use cases in data analysis

## Tools & Libraries
- Python
- Pandas
- NumPy
- Jupyter Notebook
- Excel (optional for testing)

## Purpose
- Learn to import and export datasets efficiently
- Practice file handling operations with Pandas
- Prepare datasets for analysis and machine learning
- Build strong Pandas I/O skills for real-world projects
